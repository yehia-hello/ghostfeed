# GhostFeed - Anonymous Social Platform

![GhostFeed Logo](https://i.ibb.co/9k53SFRb/Chat-GPT-Image-Apr-20-2026-10-32-28-PM.png)

## 🌟 Features

- **Anonymous Posting** - No personal data required
- **Email/Google Authentication** - Optional secure login
- **Real-time Updates** - Powered by Supabase
- **Image Uploads** - Share moments visually
- **Comments & Replies** - Engage with the community
- **Like System** - Show appreciation
- **Admin Tools** - Pin posts, delete/edit content (Dev only)
- **Premium User Badge** - Special verified status
- **Responsive Design** - Works on all devices
